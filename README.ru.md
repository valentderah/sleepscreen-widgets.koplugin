# Sleepscreen widgets

Плагин для [KOReader](https://github.com/koreader/koreader), который заменяет стандартный экран сна на настраиваемую сетку из виджетов.

![Пример виджетов](assets/screenshot-sleep-grid.png)

## Возможности

- **Типы виджетов**
  - **Текущая книга** — Название, автор и прогресс по текущей книге.
  - **Цитата** — Случайная сохранённая цитата из текущей книги.
  - **Календарь** — Показывает текущую дату.
  - **Часы** — Как аналоговые, так и цифровые.
  - **Шаблон** — Свободный текст с плейсхолдерами.
  - **Прогресс за день** — Сколько вы читали сегодня.

## Настройка KOReader

1. Включено **Settings** → **Screen** → **Sleep screen** → **Sleep screen message** → **Add custom message to sleep screen** 
2. Откройте **Container and position** и выберите **Banner** (баннер), а не **Box**.

Если что-то из этого не выполнено, KOReader покажет обычное сообщение сна.

## Установка

1. Скопируйте папку плагина в каталог `plugins` KOReader, например:  
   `koreader/plugins/sleepscreen-widgets.koplugin/`
2. Перезапустите KOReader и включите плагин в меню.
