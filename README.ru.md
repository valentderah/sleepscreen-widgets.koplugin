# Awesome Sleepscreen

Плагин для [KOReader](https://github.com/koreader/koreader), который заменяет стандартный **баннер экрана сна** на настраиваемую **сетку 6×3** из **виджетов** — карточки в стиле Frame. В ячейке не больше одного виджета; виджет может занимать **1–3 колонки**. В комплекте: часы, шаблоны, цитаты из книги, статистика сна, заголовок даты/времени, батарея, текущая книга, календарная плитка и время чтения за день.

## Возможности

- **Сетка 6×3** — До 18 слотов; **горизонтальный span** 1–3 колонки; зазоры между строками и столбцами настраиваются в оформлении баннера.
- **Типы виджетов**
  - **Шаблон** — Свободный текст с плейсхолдерами в духе KOReader (как в редакторе сообщения экрана сна).
  - **Статистика сна** — Стандартный текст сна или свой шаблон.
  - **Цитата** — Случайная сохранённая цитата из текущей книги (подсветки).
  - **Цифровые часы** — Время в формате `strftime`.
  - **Аналоговые часы** — Аналоговый циферблат со стрелками (проверялись на e-ink).
  - **Заголовок даты/времени**, **Батарея**, **Текущая книга**, **Календарная плитка**, **Чтение за сегодня** (по статистике).
- **Вёрстка почти на весь экран** — Сетка рассчитывается на полную область экрана (поверх обложки/обоев); внешние контейнеры без заливки, между карточками виден фон скринсейвера.
- **Локализация** — английский, русский и китайский (упрощённый).

## Настройки KOReader (обязательно совпасть)

Сетка плагина показывается только если выполняется **всё** ниже:

1. Включено **сообщение на экране сна**.
2. Режим контейнера сообщения — **Banner** (баннер), а не только полноэкранный текст.
3. **Тип заставки** — обложка, **случайное изображение**, **обложка документа** или **отключено** (чтобы сохранялась стандартная зона баннера).

## Установка

1. Скопируйте папку плагина в каталог `plugins` KOReader, например:  
   `koreader/plugins/awesome-sleepscreen.koplugin/`
2. Перезапустите KOReader и включите плагин в меню.
